## Objetivo

Deixar o projeto como um site "em branco", pronto para você conectar ao GitHub e editar de lá.

## O que farei

1. Substituir o placeholder em `src/routes/index.tsx` por uma página mínima e limpa (apenas um título simples tipo "Hello world" centralizado), removendo a imagem placeholder do Lovable.
2. Atualizar o `<title>` e `meta description` no `head()` da rota index e em `src/routes/__root.tsx` para valores neutros ("My Site").
3. Manter toda a estrutura base do TanStack Start (`__root.tsx`, `router.tsx`, `styles.css`) intacta para que o projeto continue buildando.

## O que NÃO farei

- Não adiciono backend, Lovable Cloud, autenticação ou banco.
- Não adiciono outras rotas, componentes ou design system customizado.
- Não conecto o GitHub por você — isso é feito pelo menu (+) → GitHub → Connect project na interface do Lovable.

## Próximo passo depois do plano

Após aplicar, é só você abrir o menu (+) no chat → GitHub → Connect project para criar o repositório e começar a editar pelo GitHub (a sincronização é bidirecional).
